# Apex Snippets

Apex Snippets is an AI-powered web app designed to help developers quickly generate common code snippets from natural language descriptions. Focus on building, not googling!.

## Features
- Generate Python list comprehensions.
- Generate JavaScript array methods.
- Generate SQL joins.
- ...and more for everyday coding tasks.

Visit `index.html` to see the product in action. Note: The `index.html` contains a front-end only simulation of the AI generation.
